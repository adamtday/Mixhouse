# Music Scene Language

## Purpose

Document the scene-native vocabulary, phrasing, and tonal cues MixHouse uses correctly (and the ones it avoids) so that copy, scripts, and captions feel like they came from inside the culture, not visiting it.

## Key Takeaways

- Scene language is precise. Mis-use signals tourism immediately.
- Use scene terms when they aid clarity, not as decoration.
- Avoid mainstream-music-press shortcuts ("EDM", "dance hits", "club bangers") in MixHouse copy.
- Defer to subject-matter advisors when uncertain.

## Core Content

### Genre Vocabulary

| Term | Use it for | Common misuse |
| --- | --- | --- |
| **Electronic music** | The broad genre. Preferred default. | "EDM" — narrow, festival-marketing term; not a synonym for the scene. |
| **House / tech house / deep house** | Specific sub-genres. Don't blend. | Lumping all four-on-the-floor under "house". |
| **Techno** | Specific sub-genre. | "EDM" or "house" substitution. |
| **Drum and bass / D&B / DnB** | Specific UK sub-genre. | Confusion with "jungle" without acknowledging the lineage. |
| **UK garage / 2-step** | Distinct historical and present sub-genre. | "Garage rock" — different genre entirely. |
| **Trance** | Specific genre with distinct sub-strands (uplifting, psy, progressive). | Used as a generic "high-energy" adjective. |
| **Afro house** | Distinct sub-genre with deep regional roots. | Treating as a sub-style of house without acknowledging origin. |
| **Amapiano** | South African sub-genre. | "African house". |
| **Hyperpop** | Production-style crossover. | Treating as a pure pop sub-genre. |

### Performance Vocabulary

| Term | Meaning |
| --- | --- |
| **DJ set** | Performed mixing of records. |
| **Live set / live PA** | Performance of original music live (synths, controllers, vocals). |
| **B2B (back-to-back)** | Two DJs sharing decks, alternating mixes. |
| **Track ID** | Identification of an unknown track played in a set. |
| **Edit / VIP / remix / dub** | Different forms of re-versioning a track. |
| **Boiler room** | Generic shorthand (lowercase) for intimate one-cam DJ format; capitalised when referring to the brand. |
| **Closing set / opening set** | Specific time-slot at a club night or festival. |
| **Peak time** | The high-intensity middle of a set. |

### Production Vocabulary

| Term | Meaning |
| --- | --- |
| **DAW** | Digital Audio Workstation. Ableton Live, Logic, FL Studio, Cubase, Bitwig. |
| **Stems** | Separated audio elements of a track. |
| **Sub** | Sub-bass; low-frequency element. |
| **Drop** | Moment of intensity in a track; structural. |
| **Hook** | Memorable musical phrase. |
| **Master / mastered** | Final processing for release. |
| **Crate / digging** | Curation / discovery of music. |

### Scene Hierarchy Vocabulary

| Term | Meaning |
| --- | --- |
| **Resident** | DJ booked regularly at a single venue or event series. |
| **Headliner** | The most-billed performer on a lineup. |
| **Promoter** | Person / company programming a club night or festival. |
| **A&R** | Label talent / repertoire function. |
| **Manager / Booking Agent** | Talent representation. |
| **Label** | Music release entity. |

### Phrases to Use

- "Played the room" — successful set, well-read crowd.
- "Reads the room" — DJ tuning a set to the audience in real time.
- "Held the floor" — kept the dance floor engaged.
- "Has a sound" — has a recognisable production identity.
- "Plays out his/her/their own productions" — performs original work.

### Phrases to Avoid

- "EDM" (in MixHouse copy unless very specifically about the festival-circuit segment).
- "Bass drop!" (used in mainstream marketing; reads cringe).
- "Sick beat" (mainstream coverage of dance music; reads tourist).
- "Club banger" (overused mainstream shorthand).
- "Underground" used as a flattering adjective for ourselves (let others say it).
- "Mainstream" used pejoratively.

### Capitalisation and Spelling

- "Drum and bass" or "D&B" — both acceptable. "Drum 'n' bass" is acceptable but inconsistent; pick one and use it consistently in any single piece.
- "Tech house" not "tech-house".
- "Boiler Room" the brand (capital) vs "boiler room" the format type (lower).
- Producer aliases respect their original capitalisation and punctuation ("Fred again..", "Peggy Gou").

### Cross-Cultural Care

When a sub-genre carries cultural-origin importance (afro house, amapiano, gqom, baile funk, footwork, kuduro), copy must:
- Acknowledge the geographic / cultural origin.
- Defer to representatives from that culture.
- Avoid framing as "discovered" by anyone outside the culture.
