# Competitive Matrix

## Purpose

Plot the comparable landscape on common axes so MixHouse's unique position is visible at a glance — and so future research deepens, rather than re-litigates, the placement.

## Key Takeaways

- No existing format combines cinematic prestige, real music creation, character residency, and franchise economics in one product.
- The matrix below shows MixHouse as the only entry checking all five capability columns.
- Singing competitions and dating shows are the closest *format* comparisons; cinematic livestream brands are the closest *aesthetic* comparisons; sports docs are the closest *prestige* comparisons.

### Read of the Matrix

- **Music creation per episode** — Only `Project Runway` is a peer on this in spirit. No music format matches.
- **Character residency** — Reality dating owns this; no music show pairs it with creation.
- **Cinematic prestige** — Sports docs and cinematic livestreams own this; no music *competition* format reaches it.
- **Real release pipeline** — Truly unique to MixHouse at scale.
- **Franchise economics** — Owned by `The Voice` and `Love Island` shape, but without the IP depth MixHouse plans.

### Aesthetic Comparison Set

For mood, look, and production reference (not direct competition):

- Cercle livestreams (cinematic, location-led).
- Boiler Room intimate-set grammar.
- Coachella livestreams (multi-camera scale).
- A24 / Hulu music documentaries (visual prestige).

### Format Comparison Set

For mechanic, character, and audience reference:

- Love Island (residency, character, franchise).
- Drag Race (creation + judging + franchise).
- Project Runway (creation cadence).
- Rhythm + Flow (music + judging at streamer prestige).

### Prestige Comparison Set

For commissioning posture and CPH:

- Drive to Survive.
- Cheer.
- Last Dance / Beckham.
- Quarterback.

### What's Missing From the Map (and Why That's the Point)

There is no row that ticks all five capability columns except MixHouse. The market gap is structural, not stylistic.
